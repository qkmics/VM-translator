This program is a part of nand2teris project, it is a Virtual Machine translator which can translate VM language to assemble language
on hack platform.

How to compile this code:
	Build a c++ project in a IDE and then add all files under /src to the project can simply compile it.
	
How to run this code:
	After compiling, run the executable file. Name of input file should be the command line argument(inputfilename.vm). Input file name 
can be in both absolute path and relative path.

What works and what doesn't work:
	This program has been tested in Visual Studio Windows 10. All .vm test files in nand2teris passed.
